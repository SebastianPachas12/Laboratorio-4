# Catalogo Web - Documentación básica

```
 Este proyecto es un pequeño catálogo web para el nivel Code 101, estos son los lenguajes 
 aprendidos durante el primer modulo.

- HTML
- CSS
- Markdown
```
## HTML
```
1. Section: Define una sección temática dentro de una página. Se utiliza para agrupar contenido relacionado.
2. Main: Representa el contenido principal del documento.
3. H1: Es una etiqueta de encabezado de nivel 1. Generalmente se usa para el título principal de la página o sección. Es importante para el SEO y la jerarquía del contenido.
4. p: Se usa para definir un párrafo de texto. Es una de las etiquetas más comunes para mostrar contenido textual.
```
## CSS
```
1. color: Define el color del texto de un elemento.
2. background-color: Establece el color de fondo del elemento.
3. padding: Controla el espacio interno entre el contenido del elemento y su borde.
4. margin: Define el espacio externo entre el elemento y otros elementos.
5. font-family: Especifica el tipo de letra que se usará para el texto.
6. font-size: Define el tamaño de la fuente del texto.
```
## Markdown
```
Markdown es un lenguaje de marcado ligero que se usa para dar formato al texto de manera sencilla y rápida, sin necesidad de escribir código HTML complejo.

Nos sirve para:
- Escribir documentos legibles y con estilo (como títulos, listas, enlaces, imágenes).
- Crear archivos README en proyectos (muy usado en GitHub).
- Tomar notas con formato en apps como Notion, Obsidian, Jupyter, etc.
- Publicar blogs o contenidos web de forma limpia y estructurada.
- Dar formato en foros, comentarios o mensajes (como en Reddit o Discord)
```
